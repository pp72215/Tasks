### **5.HTML的语法**

* HTML的元素大部分都是以开始标签为起始，以结束标签为终止
* 元素的内容是开始与结束标签之间的内容
* HTML与c语言不同，它不区分大小写，例如：<p>与<P>等价

### 6.HTML属性

* HTML的元素可以设置属性
* 属性可以在元素中夹带私货（信息）
* 属性描述在开始标签中且总是以名称/值对的形式出现

例如：

```
<a href="http://www.baidu.con">这是度娘</a>
```

* 属性值应该被包括在引号内，单双引号都可以
* [HTML标准属性](https://www.runoob.com/tags/html-reference.html)

### 7.HTML的标题与段落

**在HTML中标题是通过<h1>-<h6>标签来定义的，h1是定义最大的标题，h6是最小的标题**

* 我们可以利用<hr>标签来创建水平线，效果和下面一致

* 利用<!--巴拉巴拉-->标签可以来作为注释让我们更方便理解代码

**在HTML中俺们可以把文档分割成很多段**

### 9.HTML的超连接

**在HTML中我们可以利用<a>标签来设置超链接，来指引他人通向不同的地址（~~例如：度娘~~）**

~~~html
<a href="https://www.baidu.com" target=blank>度娘</a>
~~~

**在上面存在着href属性与target属性，href属性表示了连接的目标，target属性则定义了连接的文档在何处显示**

### 10.HTML的头部

* 在html中<head>元素包含了所有头部标签元素，类似于人类的大脑，啥都装在里面（~~除了我的脑袋里都是水~~）
* 在HTML中<title>元素定义了不同文档的标题，在浏览器的直观视角下就是这个页面的名称与你收藏后显示的名称
* 在HTML中<base>元素则描述了连接的地址/目标，表示全局的默认属性，一般不常用
* 例如<style>,<meta>，<script>等标签则在后文的CSS，JavaScript中使用较多。
